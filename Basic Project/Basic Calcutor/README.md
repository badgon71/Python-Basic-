## 🚀 Getting Started

Clone the repository:

``` bash
git clone https://github.com/YOUR_USERNAME/basic-calculator.git
```

Go to the project folder:

``` bash
cd basic-calculator
```

Install dependencies:

``` bash
pip install -r requirements.txt
```

Run the application:

``` bash
python calculator.py
```
